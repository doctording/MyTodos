
## 项目简介

python flask+mysql搭建的一个web项目，主要是一个记录个人待办事项的系统。

---


## 主要知识点

* flask简单使用

* sqlalchemy操作mysql 

* python 基础

## 主要开发环境要求

* python 2.7

* flask, sqlalchemy(数据库rom)

* mysql （见test.sql文件）


## 功能说明

### 粗版

1. 用户直接登录，未加注册功能

2. 能够显示一个登录用户的所有待办事项

3. 能够添加待办事项，但是不能修改

### 加强

* 登录，注册功能

* 数据修改功能

* 界面美观
